结构体、sort

结构体定义

struct 结构体标签
{
	变量类型 变量名;
	变量类型 变量名;
	...
	变量类型 变量名;
}结构体变量;
例如
struct node
{
	int year, month, day;
}time; // 结构体变量

struct node
{
	int year, month, day;
}time[N]; // 结构体数组

对结构体排序

例：以年为关键字从小到大排序
// 重载运算符写法
struct node
{
	int year, month, day;
	bool operator < (const node &x) const
	{
		return year < x.year;
	}
}time[N];

// 函数写法
bool cmp(node x, node y)
{
	return x.year < y.year;
}
